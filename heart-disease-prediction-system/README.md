# Heart Disease Prediction System using ANN

Machine learning project that predicts the likelihood of heart disease using Artificial Neural Networks (ANN) and patient medical data.

## Features

- Data preprocessing
- Feature scaling
- ANN model training
- Prediction system
- Confusion matrix visualization
- ROC-AUC evaluation
- Accuracy graph generation
- Model saving functionality

## Technologies Used

- Python
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Techniques

- Artificial Neural Network (ANN)
- StandardScaler
- Binary Classification

## Outputs

- Prediction CSV export
- Confusion matrix image
- Accuracy graph
- Saved trained model

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
